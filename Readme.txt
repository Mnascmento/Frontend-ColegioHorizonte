
🏫 Página Institucional Colégio Novo Horizonte




📌 Sobre o Projeto

Este projeto consiste em uma página web informativa para uma escola, apresentando todas as informações essenciais sobre a instituição. O objetivo é fornecer um ambiente digital acessível e intuitivo para alunos, responsáveis e visitantes conhecerem melhor a escola.




✨ Funcionalidades

📖 Informações institucionais: história, missão e valores;

👩‍🏫 Detalhes sobre o corpo docente;

📚 Cursos oferecidos e estrutura curricular;

🕐 Horários de funcionamento;

📅 Seção de notícias e eventos;

📷 Galeria de imagens da escola;

📞 Formas de contato e localização no mapa.




🚀 Tecnologias Utilizadas

HTML5 para estruturação da página;

CSS3 para estilização e layout responsivo;

JavaScript para interatividade e funcionalidades dinâmicas;

Bootstrap para design responsivo e componentes pré-estilizados.




🛠️ Como Executar

1. Clone este repositório;

2. Abra o arquivo index.html no navegador.
